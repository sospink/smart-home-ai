# 智慧家居系统总览（第一阶段）

## 系统目标
第一阶段目标是先完成 Dify 智能体能力闭环，不接入 Home Assistant 实控链路。

## 架构入口
- 主入口：01_main_orchestrator_chatflow
- 子流程：
  - 02_home_control_workflow（当前为 Mock）
  - 03_rag_qa_workflow
  - 04_data_analysis_workflow（当前为 Mock）

## 当前阶段边界
- 支持：意图识别、路由、结构化输出
- 不支持：真实设备控制、真实时序库查询

## 标准输出结构
```json
{
  "status": "success",
  "message": "...",
  "payload": {},
  "trace_id": "..."
}
```
