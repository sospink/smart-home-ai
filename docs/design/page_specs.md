### 1.3 功能参数

#### 统计卡片（顶部）
| 字段 | 数据类型 | 说明 |
|------|---------|------|
| 设备总数 | number | 所有注册设备数量 |
| 在线设备 | number | state ≠ unavailable |
| 离线设备 | number | state = unavailable |
| 场景数 | number | HA scene 实体数 |

#### 设备卡片
| 字段 | 数据类型 | 说明 |
|------|---------|------|
| entity_id | string | HA 实体 ID，如 `light.living_room` |
| friendly_name | string | 显示名称，如"客厅灯" |
| domain | string | 设备类型：light / switch / sensor / climate / lock |
| state | string | 当前状态：on / off / 25.6 / locked |
| attributes | object | 亮度、色温、温度设定值等扩展属性 |
| last_changed | datetime | 最后状态变更时间 |
| icon | string | 设备图标（根据 domain 自动映射） |

#### 设备分类
| domain | 图标 | 可控制 | 显示内容 |
|--------|------|:------:|----------|
| light | 💡 | ✅ 开/关 + 亮度滑杆 | 亮度 %、色温 |
| switch | 🔌 | ✅ 开/关 | 当前状态 |
| climate | ❄️ | ✅ 温度+/-、模式切换 | 设定温度、当前温度、模式 |
| sensor | 🌡️ | ❌ 只读 | 数值 + 单位（°C、%、lux） |
| binary_sensor | 🚪 | ❌ 只读 | 开/关 状态 |
| lock | 🔒 | ✅ 锁定/解锁 | 当前状态 |

### 1.4 交互设计

| 交互 | 触发方式 | 效果 |
|------|---------|------|
| **开关控制** | 点击卡片上的 Toggle 开关 | 调 API → 状态更新 → Toast 提示 |
| **亮度调节** | 拖动滑杆 | 实时调节 → debounce 300ms 发请求 |
| **温度调节** | 点击 +/- 按钮 | 温度 ±1°C → 发请求 |
| **分类筛选** | 点击左侧分类 | 卡片网格过滤显示 |
| **场景激活** | 点击场景预设按钮 | 调 scene.turn_on → 全局 Toast |
| **设备详情** | 点击卡片任意区域（非开关） | 弹出 Drawer/Modal 显示详细属性 + 历史折线图 |
| **自动刷新** | 每 10s 轮询 | 静默更新设备状态 |

### 1.5 设备详情 Drawer

| 区域 | 内容 |
|------|------|
| 顶部 | 设备名称 + 图标 + 在线状态标签 |
| 属性列表 | key-value 列表展示所有 attributes |
| 历史图表 | 最近 24h 状态变化折线图（灯=开关次数，传感器=数值曲线） |
| 操作区 | 控制按钮（开关/亮度/温度） |
| 底部 | 设备 ID + 最后更新时间 |

### 2.3 功能参数

#### 房间
| 字段 | 说明 |
|------|------|
| room_id | 房间标识（living_room / bedroom / kitchen / bathroom） |
| name | 中文名 |
| svg_path | SVG 区域路径 |

#### 虚拟设备（叠加在 SVG 上）
| 字段 | 说明 |
|------|------|
| entity_id | 对应 HA 虚拟实体 |
| position | SVG 坐标 (x, y) |
| type | light / sensor / switch / climate |
| state | 当前状态 |
| room | 所属房间 |

### 2.4 交互设计

| 交互 | 触发方式 | 效果 |
|------|---------|------|
| **查看设备状态** | 鼠标悬浮设备图标 | Tooltip 显示名称 + 状态 + 数值 |
| **控制设备** | 点击设备图标 | 弹出 Popover 控制面板（开关/亮度/温度） |
| **设备状态变化** | 实时推送 / 轮询 | 图标颜色变化（亮=黄色，灭=灰色） + 数值更新 |
| **场景切换** | 顶部场景按钮 | 批量切换设备状态 → SVG 全局动画 |
| **缩放平移** | 鼠标滚轮 / 拖拽 | SVG 可缩放移动 |

### 2.5 视觉要求

- SVG 户型图采用**简笔画风格**，白底蓝线，简洁明亮
- 设备图标为**圆形小气泡**，悬浮在对应位置
- 设备开启时图标有**呼吸光效**（微动画）
- 传感器数值直接标注在图标旁边
